This is a band social networking website for people to get more out of the listening experience.



## Info

### Requirements
* PostgreSQL 9.0+

* Ruby version
`2.3.3`

* Database creation
`bundle exec rake db:create`

* Database initialization
`bundle exec rake db:setup`

* Run the test suite with:
`bundle exec rake`
